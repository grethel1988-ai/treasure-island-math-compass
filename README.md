# 國小數學能力指標 - 高年級 - 金銀島的密碼羅盤

此專案為台灣國小數學第三學習階段（五至六年級）的能力指標題目改寫與題庫生成工具。

## 專案主題：金銀島的密碼羅盤

本專案將五六年級 24 個數學能力指標的題庫進行篩選，抽選出 50 題，並將其情境改寫為**「金銀島的密碼羅盤」**主題（包含傑克船長、航海、藏寶圖、金幣、羅盤密碼等場景），用以進行有趣的互動遊戲或課堂測驗。

## 產出檔案

- **`第三學習階段遊戲題庫.docx`**：包含 50 題以金銀島為主題改寫的國小數學五六年級選擇題庫。

## 如何執行

確保本機已安裝 Python 以及 `python-docx` 套件，並執行：

```powershell
python -X utf8 generate_game_db.py
```
